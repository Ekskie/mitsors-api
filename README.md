# Mitsors API

## Overview

Mitsors API is a NestJS REST API application that provides backend services for the Mitsors platform. The API handles user authentication, price data aggregation, and verification system for trusted traders.

## Features

- User authentication
- Price data aggregation
- Verification system for trusted traders

## Technologies

- NestJS
- Drizzle ORM / PostgreSQL
- TypeScript
- Jest
- Supertest
- Docker
- Git
- GitHub
- GitHub Actions
- GitHub Pages

## Commands

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
