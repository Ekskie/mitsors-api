## Overview

MITSORS is a **livestock price monitoring platform** that provides real-time liveweight price data for hogs across the Philippines. The platform aggregates price inputs from traders (both verified and unverified) to help users make informed decisions about livestock pricing in their region.

**Key Features:**

- Location-based price monitoring
- User role identification (Hog Raiser, Public Market Seller, Midman)
- Optional registration with social login
- Price submission for registered users
- Verification system for trusted traders
- Comprehensive user profile

- Anonymous users' name, location and roles stay on device (AsyncStorage only)
- Never sent to backend until user signs up

### 2. **Automatic Sync on Signup**

- On signup, first name, last name, location and user roles from AsyncStorage are automatically included in the signup request
- Stored in MongoDB user profile for registered users

### 3. **User Role Selection**

- Users can select multiple roles (Hog Raiser, Public Market Seller, Midman)
- At least one role must be selected
- Roles can be updated later in profile settings

### 4. **Verification Benefits**

- Verified traders' prices appear in the "Verified Average" calculation
- Higher trust and visibility in the community

### 5. **Profile Access**

- Verified users can view their verified documents (ID, business permit, phone number) in their profile
- All users can view their submission history

### 6. **Optional Signup**

- Users can view prices without registering
- Registration is required only to submit prices and access verification

### 7. **Data Storage Strategy**

- **Anonymous**: First name, last name, location and roles in AsyncStorage only
- **Registered**: First name, last name, location and roles in both AsyncStorage and MongoDB
- **Price Data**: All stored in MongoDB via NestJS backend
- **Verification**: Documents stored in MongoDB GridFS or cloud storage
